Place Google service account key here for backend runtime.

Expected file name:
- `google-service-account.json`

Configured via:
- `GoogleCloud:CredentialPath` in `appsettings.json`
- Default value: `credentials/google-service-account.json`

For server deploy, prefer environment variable:
- `GOOGLE_APPLICATION_CREDENTIALS=/absolute/path/to/google-service-account.json`
